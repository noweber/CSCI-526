1. Must have Python3 and venv on the laptop/system already.
2. Move the complete heatmap folder to any desired <path>.
3. Change the directory to the <path> 
4. Run the below commands from 'inside' the heatmap directory.

source heat/bin/activate   (Make sure CLI prompt changes to 'heat' virtual env)
pip install -r requirements.txt
python test.py

5. source heat/bin/activate has to be run ONLY ONCE to setup the virtual environment named "heat".
